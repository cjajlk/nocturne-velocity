## 🌌 Concept

Nocturne Velocity est un arcade spatial à l’ambiance nocturne,  
calme et tendue.

Le joueur incarne un pilote évoluant à travers des mondes progressifs,  
avec système de modules, drones, prestige et mode Hardcore.

L’expérience vise :
- une immersion atmosphérique
- une tension maîtrisée
- une progression long terme

---

## 🚀 Fonctionnalités actuelles

- Gameplay arcade spatial
- Mode Normal
- Mode Hardcore (déblocage conditionnel)
- Système de garage (vaisseaux, modules, drones)
- Profil détaillé (Stats / Combat / Prestige / Équipement)
- Options complètes (Audio / Graphismes / Gameplay / Interface)
- Footer versionné global
- Architecture extensible pour mondes + boss
